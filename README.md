# EVE Online Resource Price Forecasting System
Аналитическая система для прогнозирования изменений стоимости ресурсов на рынке EVE Online

# 1. Краткое описание программного обеспечения
Данный проект представляет собой аналитическую систему, разработанную для прогнозирования цен ключевых ресурсов в онлайн игре: MMORPG EVE Online. Система использует рыночные данные для построения и оценки прогностических моделей с целью определения потенциального роста цен.

Технологический стек: Python (с использованием библиотек Pandas, NumPy, Scikit-learn/Statsmodels).
